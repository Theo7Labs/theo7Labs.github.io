01 — Featured Build

A deployed
serverless AI workflow.

An end-to-end serverless AI document workflow on AWS — from PDF ingestion through structured extraction to a working frontend. Implemented with AI-assisted coding, deployed by hand, and documented as a single working system.

Flagship Build

THEO7 AI Document Intelligence Pipeline

theo7-docai.lovable.app ↗

A deployed serverless AI document-processing workflow on AWS. Users upload PDFs to S3, which triggers a Lambda function that sends content to Amazon Bedrock (Claude) for structured extraction. Results are persisted in DynamoDB and exposed through an API Gateway REST endpoint consumed by a React frontend. The full stack is deployed as code with AWS SAM, with IAM least-privilege scoped across services.

Implemented with AI-assisted coding — ChatGPT and Claude helped generate and refine the code. The hands-on contribution is in the AWS configuration, service integration, deployment, and understanding how the pieces fit together end to end.

Architecture Flow

S3→ Lambda→ Bedrock · Claude→ DynamoDB→ API Gateway→ React Frontend
