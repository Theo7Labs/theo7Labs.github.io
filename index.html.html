01 — What I Built

Proof of work.
Not proof of intent.

Flagship Project

THEO7 AI Document Intelligence Pipeline

theo7-docai.lovable.app ↗

A fully serverless, event-driven document processing pipeline built from scratch using AWS SAM — deployed across 7 phases. Documents are uploaded to S3, trigger EventBridge rules, processed by Lambda, analyzed by Textract and Bedrock, and surfaced through a React frontend via API Gateway. No managed services doing the heavy lifting — every layer architected intentionally.

Architecture Flow

S3→ EventBridge→ Lambda→ Textract→ Bedrock→ API Gateway→ React Frontend
